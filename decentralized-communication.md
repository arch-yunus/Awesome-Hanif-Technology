# ?? Merkeziyetsiz ve Fıtri İletişim (Decentralized Darusselam)

Sermaye devlerinin (Amon Rahiplerinin) iletişim tekelini kıran "Asa" teknolojileri.

## ?? P2P (Noktadan Noktaya) Haberleşme Protokolleri

### [Signal](https://github.com/signalapp)
- **Açıklama:** End-to-end şifreli, açık kaynak mesajlaşma uygulaması
- **Hanif Test:** Ahlaki Bütünlük (gizlilik odaklı), Bağımsızlık (reklam yok)
- **Lisans:** GPL-3.0
- **Platform:** iOS, Android, Desktop

### [Matrix](https://github.com/matrix-org)
- **Açıklama:** Açık, merkeziyetsiz iletişim protokolü (federated)
- **Hanif Test:** Bağımsızlık (federated, tek sunucu bağımlılığı yok), Ahlaki Bütünlük
- **Lisans:** Apache-2.0
- **Platform:** Tüm platformlar

### [Element (formerly Riot)](https://github.com/vector-im/element-web)
- **Açıklama:** Matrix protokolü için açık kaynak istemci
- **Hanif Test:** Bağımsızlık (federated), Ahlaki Bütünlük
- **Lisans:** Apache-2.0
- **Platform:** Web, Desktop, Mobile

### [Session](https://github.com/loki-project/session-desktop)
- **Açıklama:** Merkeziyetsiz, metadata gizleyen mesajlaşma
- **Hanif Test:** Bağımsızlık (tamamen P2P), Ahlaki Bütünlük (maksimum gizlilik)
- **Lisans:** GPL-3.0
- **Platform:** iOS, Android, Desktop

### [Briar](https://github.com/briarproject/briar)
- **Açıklama:** İnternet olmadığında Bluetooth/Wi-Fi Direct ile çalışan P2P mesajlaşma
- **Hanif Test:** Bağımsızlık (tamamen P2P), Zararsızlık (internet bağımlılığı yok)
- **Lisans:** GPL-3.0
- **Platform:** Android, Desktop

### [Jami](https://github.com/savoirfairelinux/ring-project)
- **Açıklama:** Merkeziyetsiz video konferans ve mesajlaşma (SIP tabanlı)
- **Hanif Test:** Bağımsızlık (P2P), Ahlaki Bütünlük
- **Lisans:** GPL-3.0
- **Platform:** Linux, macOS, Windows, iOS, Android

### [Tox](https://github.com/tox/toxcore)
- **Açıklama:** Tamamen P2P, sunucusuz anlık mesajlaşma protokolü
- **Hanif Test:** Bağımsızlık (sunucusuz), Ahlaki Bütünlük
- **Lisans:** GPL-3.0
- **Platform:** Tüm platformlar

## ?? Mesh Networks (Topluluk Destekli İnternet)

### [Babel](https://github.com/JulienBabel/Babel)
- **Açıklama:** Yönlendirme protokolü - mesh ağlar için
- **Hanif Test:** Bağımsızlık (ISP bağımsız), Zararsızlık (yerel altyapı)
- **Lisans:** MIT
- **Platform:** Linux/Unix

### [cjdns](https://github.com/cjdelisle/cjdns)

- **Hanif Test:** Bağımsızlık (ISP bağımsız), Ahlaki Bütünlük

- **Platform:** Linux, macOS, Windows

- **Lisans:** LGPL-3.0
- **Platform:** Go (çoklu platform)

### [Libremesh](https://github.com/libremesh)

### [Commotion Wireless](https://github.com/opentechinstitute/commotion)

- **Hanif Test:** Bağımsızlık (topluluk ağı), Zararsızlık

### [NYU Mesh](https://github.com/nyu-mesh)

- **Platform:** Linux
