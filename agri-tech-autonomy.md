# ?? Hanif Tarım ve Fiziksel Çapalar (Agri-Tech & Autonomy)

*Yusuf Personası'nın özmesi giyip sahaya inmesi:* Dijital dünyadan fiziksel otonomiye geçiş.

## ?? Akıllı Çiftlik Yönetim Sistemleri

### [FarmOS](https://github.com/farmOS/farmOS)
- **Açıklama:** Açık kaynak tarım yönetim sistemi - tohum, hasat, ekipman takibi
- **Hanif Test:** Bağımsızlık (self-hosted, bulut bağımlılığı yok), Zararsızlık
- **Lisans:** GPL-2.0
- **Platform:** Web (Drupal tabanlı, self-hosted)

### [FarmBot](https://github.com/FarmBot)
- **Açıklama:** Açık kaynak, CNC tabanlı otomatik bahçe robotu
- **Hanif Test:** Dokunulabilirlik (fiziksel üretim), Bağımsızlık
- **Lisans:** GPL-3.0, MIT
- **Platform:** Hardware + Web

### [Trellis](https://github.com/Trellis-Platform)
- **Açıklama:** Açık kaynak tarım aracı yönetim sistemi
- **Hanif Test:** Bağımsızlık (self-hosted), Dokunulabilirlik
- **Lisans:** GPL-3.0
- **Platform:** Web (Django)

### [OpenFarm](https://github.com/openfarmcc/OpenFarm)
- **Açıklama:** Bitki yetiştirme rehberleri ve topluluk veritabanı
- **Hanif Test:** Bağımsızlık (topluluk odaklı), Dokunulabilirlik
- **Lisans:** MIT
- **Platform:** Web

## ?? Tohum ve Genetik Kaynakları

### [Open Source Seed Initiative](https://www.opensourceseed.org/)
- **Açıklama:** Patentli tohumlara karşı açık kaynak tohum lisanslama sistemi
- **Hanif Test:** Ahlaki Bütünlük (tekel karşıtı), Bağımsızlık
- **Lisans:** Özel OSSI lisansı
- **Platform:** Fiziksel (tohum)

### [The Global Biodiversity Information Facility (GBIF)](https://www.gbif.org/)

- **Hanif Test:** Ahlaki Bütünlük (bilgi paylaşımı), Bağımsızlık
- **Lisans:** CC0, CC-BY
- **Platform:** Web API

- **Hanif Test:** Ahlaki Bütünlük (genetik çeşitlilik koruma), Dokunulabilirlik
- **Lisans:** Çeşitli
- **Platform:** Web

### [Seed Savers Exchange](https://github.com/SeedSavers)
- **Açıklama:** Tohum takas ve koruma ağı
- **Hanif Test:** Ahlaki Bütünlük (tekel karşıtı), Dokunulabilirlik
- **Lisans:** Topluluk odaklı
- **Platform:** Fiziksel + Web
